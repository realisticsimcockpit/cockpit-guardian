# Code of Conduct

Cockpit Guardian is a practical engineering project for simracing cockpit
reliability. Contributions and discussions should stay respectful, focused, and
useful.

## Expected Behavior

- Be clear and constructive.
- Assume good intent.
- Keep technical criticism about code, behavior, or documentation.
- Respect privacy when logs contain device serial numbers or local paths.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Posting private credentials or secrets.
- Publishing exploit details for unresolved security issues.

## Enforcement

Project maintainers may edit, hide, or remove comments and issues that violate
this code of conduct. Repeated abuse can result in a block from the project.
