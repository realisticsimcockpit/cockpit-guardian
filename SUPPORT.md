# Support

Use GitHub Issues for support, bug reports, and feature requests:

https://github.com/realisticsimcockpit/cockpit-guardian/issues

When reporting cockpit/device issues, include:

- Windows version.
- Cockpit Guardian version.
- Device type, for example wheelbase, pedals, shifter, handbrake, DDU, Arduino,
  ESP, wind simulator, or button box.
- Whether the device is Required or Optional.
- Expected COM port or joystick order if relevant.
- A log export from the Logs tab when available.

Do not post private credentials, GitHub tokens, or personal access tokens.
