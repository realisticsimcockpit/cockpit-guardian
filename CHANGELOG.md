# Changelog

All notable changes to Cockpit Guardian are documented here.

## 0.1.0 - 2026-06-13

### Added

- Initial PySide6 desktop application.
- Dashboard, USB Health, Logs, Settings, and Advanced / Debug tabs.
- Save Configuration, Check Now, Restore, and Rollback Last Restore actions.
- JSON snapshot and settings storage.
- Serial/COM, HID, joystick order, USB Health, SimHub, and software detection services.
- Restore engine with backup creation before restore attempts.
- Windows tray icon and context menu.
- Integration notices for SimHub, Arduino, ESP boards, and Windows restore behavior.
- Nuitka standalone build and Inno Setup installer scripts.
- GitHub Actions CI on Windows with Python 3.12.
- Dashboard export/import of portable configuration backups for Windows reinstall recovery.
- Best-effort USB generation display per Dashboard device.
